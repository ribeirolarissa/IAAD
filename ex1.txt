Bancos de dados são o conjunto de arquivos relacionados entre si com registros sobres coisas, lugares ou pessoas, coleções organizadas de dados relacionados a fim de criar uma informação e gerar maior eficiência em pesquisas. Já o Sistema de gerenciamento de Banco de Dados, como sugere o nome, é o conjunto de softwares responsáveis pelo gerenciamento de um banco de dados, tendo como objetivo principal retirar da aplicação cliente a responsabilidade de gerenciar a organização, o acesso e a manipulação dos dados.
